Contact information should be formatted using best practices for usability on both desktop and mobile devices:

  • Street addresses should be coded as hyperlinks to Google maps or using the coordinates and the geo: protocol of an anchor element for use with GPS applications on mobile devices.
  • Telephone numbers should be coded with click-to-call functionality by wrapping them in a par of anchor (<a></a>) tags with an href attribute containing the tel: protocol that specifies.
  • Email addresses should also be formatted as hyperlinks using the mailto protocol in the href attribute value of the anchor tag.
